char  Delim 
)
staticprotected

Splits string based on deliminating character.

SplitString takes in a string s and a deliminating char d. A vector of strings is generated by splitting s anywhere it contains d. The character d is therefore not contained in the strings resulting from a split.

Parameters
StringThe string to be split.
DelimThe deliminating character to split String with.
Returns
Vector of strings from splits.
