Detailed Description

This class represents the bare minimum necessary to display a window.

Creating an instance of this class will set up and run all the necessary components.

This is the virtual windows procedure that is called by the actual WindowProc.

Classes requiring special handling of messages should override this function and provide their own implementations. The function as implemented in this class just handles basic messages.
