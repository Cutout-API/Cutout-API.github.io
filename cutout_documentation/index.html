Introduction

The CutoutAPI library is a library for generating mesh data from SVG files to enable direct rendering of them via the GPU. This document exists as the developer's manual for the API itself.

FAQs

Is this the final version of this user manual document?
This API is in active development. Although the core functionality and structure of the API is set, many function calls have yet to be deicded. Therefore, please treat this document as a "living document". However, we plan to deliver an Alpha version of the API soon.

What operating systems do you plan on supporting?
At first we plan on supporting at least Windows 10. Later on, we will look into Linux based operating systems as well as Linux.

What graphics platforms do you plan on utlizing?
We plan on supporting Direct X 12 first and Vulkan later on.

What feature set do you plan to support?
At the start, we plan on supporting most of SVG's path logic. We also plan on having basic color support.
