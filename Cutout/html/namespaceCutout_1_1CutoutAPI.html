Cutout::CutoutAPI::DX12SVGMesh Cutout::CutoutAPI::CreateDX12SVGMesh ( ID3D12Device *  Device,
std::string  File 
)

Applications should call this function during initialization to create the SVGMesh and associated DX12 components.

Given a file path, File, representing either a .svg or .raw file this function will use that file to generate a SVGMesh object. The MeshCreator library will be used to generate the actual mesh. The DX12 device, Device, will be queried initially to determine resolution information and that data will be passed along with the file into the MeshCreator library. Once the mesh is created a DX12SVGMesh object is created and filled up with the necessary data. All the mesh data will be uploaded to the GPU during this stage and the various addresses will be stored as a list of DX12Mesh objects.

This function should only ever be called once per SVG file per application lifetime.

Parameters
DeviceThe DX12 device that will be used to render the mesh.
FileThe path to a file containing an SVG or RAW.
Returns
DX12SVGMesh that represents the SVG file provided.
DX12PROCESSOR_API int Cutout::CutoutAPI::DestroyDX12SVGMesh ( DX12SVGMesh *  Mesh)

Applications should call this once they are done using an DX12SVGMesh object.

Once an application is done using a DX12SVGMesh object they need to call this function to clean up the various GPU resources being referenced by it. Once this function has been called the DX12SVGMesh object is considered destroyed and should no longer be used.

This function should only ever be called once per SVG file per application lifetime.

Parameters
MeshThe DX12SVGMesh to be destroyed.
Returns
0 for success or 1 for failure.
DX12PROCESSOR_API int Cutout::CutoutAPI::RenderDX12SVGMesh ( DX12SVGMesh *  Mesh,
ModelViewProjection  Transforms,
ID3D12CommandQueue *  Queue 
)

Applications should call this to render the SVG mesh during their rendering cycle.

After calling CreateDX12SVGMesh and before calling DestroyDX12SVGMesh the DX12SVGMesh can be used to render the SVG mesh into a frame buffer. The location of the mesh in the scene are provided through the transforms. The resulting command list is applied to the supplied command queue.

This function can be called multiple times per frame to render the same mesh in different places in the scene.

Parameters
MeshA mesh generated using CreateDX12SVGMesh.
TransformsThe model, view, and projection matrices.
QueueThe command queue to execute the command list through.
Returns
0 for success or 1 for failure.
DX12PROCESSOR_API int Cutout::CutoutAPI::SaveSVGMeshToRawFile ( DX12SVGMesh *  Mesh,
std::string  NewRawFile 
)

Applications should call this function during initial install or first application start to generate RAW files that will load up more quickly that SVG files.

After calling CreateDX12SVGMesh the DX12SVGMesh can be outputted in an optimized format referred to as RAW. Using the RAW files to load up SVG assets will be faster than using the SVG files themselves.

Parameters
MeshThe mesh object to save to a RAW file.
NewRawFileThe path to save the new raw file to.
Returns
0 for success or 1 for failure.
