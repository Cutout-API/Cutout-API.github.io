Cutout Documentation

Description

The CutoutAPI library is a library for generating mesh data from SVG files to enable direct rendering of them via the GPU. It is comprised of three parts:

  • ComputeEngine - A GPU utilizing library that MeshCreator rely's on to do highly parralized tasks.
  • MeshCreator - The mesh generation library.
  • CutoutAPI - The API that MeshCreator is accessed through.

ComputeEngine

This library is not under development currently and is not a part of Senior Design. The ComputeEngine library will be focused on acheiving faster mesh generation when used with MeshCreator. It will acheive this by using the GPU. We are planning on this library using compute shaders to access the GPU.

MeshCreator

The MeshCreator library is a serial library that generates a mesh from an SVG file. Eventually this library will use the ComputeEngine library to use the power of the GPU to optimize mesh generation.

CutoutAPI

The API for cutout is broken into three parts: CutoutDX12API, CutoutObjAPI, CutoutVulkanAPI. The CutoutVulkanAPI is not part of Senior Design. Each of these API's are meant to support a different graphics API or workflow. They define methods for handling the set-up of basic structures for the various system they are mirroring. They all use MeshCreator to generate the meshes from the SVG files.
